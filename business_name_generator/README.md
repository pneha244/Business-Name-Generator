# Business Name Generator

Simple Flask-based Business Name Generator.

Run locally:

1. Create a virtual environment and activate it.
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
python app.py
```

Open http://127.0.0.1:5000 in your browser.

Acceptance criteria: generates 1..N relevant, unique names; validates input; supports styles and regenerate.

This app uses a simple AI-style scoring heuristic to rank generated names by relevance and semantic fit.
